#TP Profesional Frontend

##To start the app:

### Requirements
* [NodeJS](http://nodejs.org/) (with [NPM](https://www.npmjs.org/))
* [Bower](http://bower.io)
* [Gulp](http://gulpjs.com)

####Install dependencies
```
$ npm install
$ bower install
```

####Run application 
```
$ gulp serve
```
